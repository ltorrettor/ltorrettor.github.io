Quod Erat Demonstrandum

How to play
To apply a rule of inference, click on two expressions and a popup menu will appear. That popup will list all rules applicable to your selection, where you can choose which one to apply by clicking on it.
To select only one expression, click on it twice.
Note that when selecting only one expression, you can be specific as to which sub-expression to select (which is crucial for rules of replacement).
Inference rules must be applied exactly, so if you're having trouble applying a rule you think should be possible, you may need to use Double Negation or Commutativity first.
Please note
This game is currently playable and you should be able to prove all puzzles, however it is currently incomplete. The red "X" button (for removing lines) and the drag handles (for re-arranging lines) do not work.
Also, nothing in particular happens when you complete a proof, so for now you're responsible for congratulating yourself.
